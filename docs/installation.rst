============
Installation
============


Stable release
--------------

To install Betrayal at the House on the Hill, run this command in your terminal:

.. code-block:: console

    $ pip install betrayal_at_the_house_on_the_hill

This is the preferred method to install Betrayal at the House on the Hill, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Betrayal at the House on the Hill can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/JamesG419/betrayal_at_the_house_on_the_hill

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/JamesG419/betrayal_at_the_house_on_the_hill/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/JamesG419/betrayal_at_the_house_on_the_hill
.. _tarball: https://github.com/JamesG419/betrayal_at_the_house_on_the_hill/tarball/master
